## Quick Start（とりあえずこれをやれば動く、という最短手順）

1. 前提条件を満たしていることを確認する（Node 数、ディスクなど）
2. Helm リポジトリを追加する
3. Longhorn をインストールする
4. Web UI にアクセスしてボリュームを1個作成する

※ 実際のコマンドはこの下にコードブロックで書く：

（例）
- `helm repo add ...`
- `helm upgrade --install ...`

---

## 詳細手順

### 1. 前提条件
- Kubernetes クラスタ：v1.xx 以上
- Node：3台以上推奨
- コンテナランタイム：containerd など

### 2. インストール手順
1. Helm リポジトリ追加
2. Namespace 作成
3. Helm install / upgrade

（ここに実際のコマンドを書く）

### 3. 動作確認
- Pod / PVC / PV の状態確認
- Web UI へアクセス
- テスト用ボリューム作成

---

## トラブルシュート

| 症状 | 想定原因 | 対処 |
|------|----------|------|
| PVC が Pending のまま | StorageClass が設定されていない | Deployment / StatefulSet の `storageClassName` を修正 |
| Longhorn UI にアクセスできない | NodePort / Ingress 未設定 | Service/Ingress 設定を見直し |
